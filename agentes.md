# agentes.md — Mining Productivity 2.0
## JEBI Hackaton 2026 | Operator Efficiency Profile (OEP) MVP v2

Este repo implementa un MVP funcional para calcular un Operator Efficiency Profile usando:
- video estéreo left/right
- IMU en `.npy`
- generación de reporte JSON
- resumen narrativo
- visualización PNG
- dashboard HTML offline

La versión está aterrizada para correr offline y degradar con gracia si algún video falla.
